## 📅 搭建日志

### 2026年3月

#### Mac mini 到货 📦

新购的Mac mini终于到了！专门用来跑自托管服务。配置如下：
- M4 chip
- 16GB RAM
- 512GB SSD

第一件事就是安装基础开发环境：Node.js 22、Git、Docker。

#### OpenClaw 初体验 🦞

按照官方文档完成了基础安装：

```bash
npm install -g openclaw
openclaw init
openclaw dev
```

第一次运行就遇到了坑：Node.js版本太低。升级到22+后解决。

#### 第一个AI消息 🎉

配置好Claude API后，发送了第一条测试消息。成功了！

这时候才真正感受到自托管AI的魅力 - 所有数据都在本地，完全掌控。

## 📝 心得体会

### 关于自托管

把AI助手跑在自己机器上，最大的感受是**安全感**。不用担心聊天记录被上传，不用担心隐私泄露。虽然麻烦一点，但值得。

### 关于OpenClaw

OpenClaw的设计真的很优雅，配置文件清晰，扩展性强。连接多个聊天平台也非常方便。目前已经集成了Telegram，下一步准备试试WhatsApp。

### 关于节奏

没有给自己设定"Day 1-3"这种时间表。想折腾就折腾，累了就停。个人项目嘛，开心最重要。
